# ProtocolManager
## 一款基於HslCommunication的擴展插件 by黃御德

## 簡介
### ProtocolManager是hslCommunication的讀寫器擴展，打破了傳統讀取陣列一個一個對應的窘境，目前由本人開發大部分的功能，基本上可以單獨對每一個暫存器編上自己定義的標籤，包括讀以及寫，目前僅支持NetworkDeviceBase基類的擴展。
